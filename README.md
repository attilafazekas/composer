# Composer

A music writing web app for guitarists and songwriters.

## Features

- **Neck** — Interactive guitar fretboard with scale/chord highlighting. Supports any tuning including Drop B, Drop C, Open G, DADGAD, and custom.
- **Chords** — Diatonic chord grid for any key + scale. Build progressions, copy to clipboard, pick from common patterns.
- **Lyrics** — Write song sections (verse, chorus, bridge…) with a built-in Claude AI assistant for suggestions, rewrites, and rhymes.
- **Structure** — Drag-and-drop section timeline to arrange your song flow.

## Setup

1. Open `index.html` in any browser — no server or build step needed.
2. Click ⚙ to add your [Anthropic API key](https://console.anthropic.com) (needed for the Claude lyrics assistant).

## GitHub Pages

Enable Pages in your repo settings → Source: `main` branch, root folder.

## Tech

Vanilla HTML/CSS/JS — no dependencies, no build step.
